Steps to setting up the application.

1) Go to the github repository and clone it to the directory of your choice.
2) Open the terminal and navigate to the root directory of the project.
    2.1) Run "cd ui" to get to the ui directory.
    2.2) Run the command: yarn start
3) Open another terminal window/tab.
    3.1) Run "cd server" to get to the server directory.
    3.2) Run the command: yarn dev
4) Steps to be updated.

*** Fair warning, this is for testing the application on the local machine. You might not have all the components configured or installed.
